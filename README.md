Proyek ini adalah aplikasi dasar untuk sistem autentikasi pada perangkat mobile Android. Fungsi utama aplikasi ini adalah pengelolaan sesi pengguna, validasi input, dan transisi antar activity.

->Fitur Utama
Sistem Login & Register: Alur autentikasi pengguna dan validasi jika inputnya ngga sesuai.

Validasi Form: Pengecekan input email dan password secara real-time.

UI Responsif: Tampilan modern menggunakan Jetpack Compose dan XML Layout.

-> Stack Teknologi
Bahasa: Kotlin

UI Framework: XML (Layout Editor) & Jetpack Compose

Min SDK: API 30

->Komponen Utama
MainActivity.kt: Logika utama untuk menangani interaksi pengguna dan navigasi setelah login berhasil.

activity_main.xml: Struktur tata letak antarmuka (UI) untuk halaman login, termasuk EditText dan Button.
